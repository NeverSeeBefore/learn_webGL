## 向量
                               →
向量： 有大小，有方向的量，也称矢量 OA
标量： 只有大小，没有放线  OA


→            →
OA = (x1,y1) OB = (x2, y2)

和向量
→    →                         →
OA + OB = (x1 + x2, y1 + y2) = OB

差向量
→    →                         →
OA - OB = (x1 - x2, y1 - y2) = BA

点乘 内积、数量积
结果是 数值
→    →
OA · OB = x1 * x2 + y1 * y2 =|A||B|Cos(θ)

叉乘 外积、向量积
叉乘的结果是个向量，方向在z轴上

→    →
OA * OB = |A||B|Sin(θ)


→    →     →      →
OA · OB = |OA| * |OB| * cos（θ）
模 向量的大小
 →
|OA| = √(x1^2 + x2^2)

每个位置对应相加
要求 两个加数的 行列数 相同，否则不能相加。

乘法
满足结合律分配律

数乘
矩阵的每个位置和这个数相乘
满足交换律
